아이디를 잊어버린 경우
  1. 처음 설정한 기기(PC·폰)에서 앱을 열면 아이디가 자동으로 표시돼요 — 상단 파란 뱃지를 확인하세요.
  2. 클라우드(Supabase)에 연동돼 있다면, 관리자에게 요청해 Supabase → Table Editor → worklog 테이블의 key 목록에서 내 아이디를 찾을 수 있어요. 아이디_all_data 형태로 저장돼 있습니다.
  3. 아이디를 찾지 못하면 새 아이디로 다시 시작해야 해요. 이전 데이터는 클라우드에 보존되어 있으니 아이디를 찾으면 복구 가능해요.

Supabase 설정 방법 (처음이라면 클릭)
  1. supabase.com 에서 무료 계정 생성 후 New Project 클릭
  2. 프로젝트 생성 후 왼쪽 메뉴 SQL Editor 클릭 → 아래 코드 붙여넣고 실행
  3. CREATE TABLE worklog (key TEXT PRIMARY KEY, value TEXT, updated_at TIMESTAMPTZ DEFAULT NOW());
  4. 왼쪽 메뉴 Project Settings → API 에서 Project URL과 anon public key 복사
  5. 위 입력칸에 붙여넣고 저장 및 동기화 클릭
  6. 모든 팀원이 동일한 URL과 Key를 입력하되, 각자 다른 개인 아이디를 사용하면 데이터가 분리됩니다
